# Example0011_SnapGridInfiniteDragon_SnapGridOnly

A minimal snap-to-grid demo built from Example0009_Crosshairs.

## What it shows
- Infinite background grid
- Crosshairs overlay
- Snap-to-grid behavior derived from Example0008_SkeletalApplicationBasic
- Pan and zoom support

## Controls
- **F9** toggle grid snap on/off
- **X** cycle crosshair mode
- **B** cycle background preset
- **R** reset view
- **Right mouse drag** pan
- **Mouse wheel** zoom
