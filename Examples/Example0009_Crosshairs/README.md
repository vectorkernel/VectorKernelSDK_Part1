# Example0009_Crosshairs

Standalone crosshairs demo built from the infinite-grid sample.

## What it shows

- infinite world-space grid
- screen-space Crosshairs overlay from `RenderCoreLib/Crosshairs`
- separate scene and overlay VBOs
- `WM_MOUSELEAVE` support so the overlay hides when the cursor leaves the client area

## Controls

- Move mouse: move crosshairs
- Right mouse drag: pan view
- Mouse wheel: zoom
- `X`: cycle crosshair mode
- `B`: cycle background preset
- `R`: reset view
